# 项目模拟面试题库（面试官视角）

> 基于 Vue3 + Vite + TailwindCSS 通用型项目，从面试官角度整理的完整问题库

---

## 📋 项目信息（技术栈与亮点）

### 技术栈
- **框架**：Vue3 / Vite / TailwindCSS
- **工具库**：VueUse / VeeValidate / GSAP / Cropperjs / ali-oss / driver.js

### 项目定位
集 **42 种前台业务模型**实现方案 + **15 种中台通用组件**物料库于一身的通用型项目

### 核心亮点
- ✅ **Vite 工程化**：多套路由表配置、SVG 图标自动化注册、通用组件与指令自动化注册
- ✅ **物料库**：16 个通用组件（Infinite、Waterfall、Popup、Popover、Confirm、Dialog、Countdown…）
- ✅ **性能优化**：瀑布流/长列表 + 懒加载 + 渐进式图片；堆内存同比 -35%，FCP 平均 1s，Lighthouse +33%
- ✅ **登录架构**：SliderCaptcha 行为验证 + VeeValidate 自定义校验 + 登录鉴权
- ✅ **头像裁切**：Cropperjs 适配 PC/移动端差异，上传阿里云 OSS 并更新头像
- ✅ **H5 体验**：路由动效 + 虚拟任务栈缓存 + 滚动位置记录，保证切换丝滑
- ✅ **第三方能力**：QQ/微信扫码登录、支付宝 PC 扫码 & 移动端调起、微博分享、兔小巢反馈
- ✅ **特别功能**：暗黑主题、新手引导、文件下载、指定 DOM 全屏、动画（GSAP）
